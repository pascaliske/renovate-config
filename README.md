## Usage

`renovate.json`

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["pascaliske/renovate-config"]
}
```

For more information visit the [Renovate documentation](https://docs.renovatebot.com/config-presets/).
